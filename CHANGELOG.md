## [1.2.0] 2025-07-02 · 活化溯源与反伪科普机制

### 新增
- **活化溯源模块**：生成结果顶部新增"活化溯源"卡片，展示典籍、篇目、版本、时代、参考注本
- **原文锚定功能**：白话解读支持展开"原文锚点"，逐句标注对应原文与置信度（高/中/低）
- **置信度评分**：以可视化进度条展示生成内容的高/中/低置信占比
- **改编声明**：对超出原文范围的艺术加工内容给出明确声明
- **分镜原文依据**：每个短视频分镜卡片底部标注对应的原文依据
- **可溯源分享卡**：文创分享卡底部自动附加典籍出处水印
- **引用格式复制**：一键复制标准引用格式，便于创作者标注来源
- **项目修改记录**：新增 `CHANGELOG.md`，同步记录每次改动

### 优化
- 内置四部古籍补充权威版本与参考注本信息（朱熹《诗集传》、杨伯峻《论语译注》等）
- `showcase.html` 作品展示页突出"反伪科普"三层约束理念
- `README.md` 同步更新功能描述与反伪科普设计说明

### 修复
- 修复 README 中 GitHub Pages 链接的死链问题
- 修复 `showcase.html` 空文件问题

## [1.1.0] 2025-07-01 · 表演型作品展示与视觉升级

### 新增
- **表演型首屏**：`showcase.html` 加载后自动演示古文竖排展开 → 白话解读打字机 → 分镜胶片翻出
- **古籍书影选卡**：内置典籍卡片改为仿古籍封面设计，含竖排书名与朝代签
- **二创边界可视化**：绿区/红线分区展示，红线区加半透明"禁"字印章
- **横向分镜时间轴**：60 秒短视频分镜改为可横向滑动的胶片卷轴
- **文创分享卡**：9:16 竖排卡片，含朱砂边框、印章落款，支持下载 PNG

### 优化
- 整体视觉升级为"新中式数字古籍"风格：纸张纹理、水墨晕染、书法字体
- 导航栏、Hero、结果卡片、页脚统一古籍气质

### 修复
- 移除与 GitHub 官方 Static HTML workflow 冲突的旧部署配置
- 成功部署到 GitHub Pages
