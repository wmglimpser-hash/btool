# 古籍活化工具站 - 规格文档

## 1. Concept & Vision

一个将古籍图文素材转化为可传播内容的工具站，为文博、教育、科普和短视频创作者提供一站式古籍内容活化解决方案。

产品调性：**古籍今用，以今释古** —— 界面承载古籍气质，内容输出现代传播力。

## 2. Design Language

### 美学方向
新中式数字古籍 —— 融合传统水墨意境与极简现代布局，以"数字古籍"为核心视觉隐喻。

### 色彩系统
```
--ink-black: #1a1a1a       /* 墨黑 - 主背景/标题 */
--rice-paper: #faf8f5      /* 宣纸白 - 主背景 */
--cinnabar: #c94043        /* 朱砂红 - 强调/按钮 */
--bronze: #8b7355          /* 青铜色 - 次级元素 */
--ink-wash: #4a4a4a        /* 墨色 - 正文 */
--gold-accent: #b8860b     /* 暗金 - 装饰高亮 */
--mist-gray: #e8e6e3       /* 烟灰 - 卡片/分割 */
```

### 字体
- 标题：`"Noto Serif SC", "Source Han Serif CN", serif`
- 正文：`"Noto Sans SC", "Source Han Sans CN", sans-serif`
- 装饰/标签：`"Ma Shan Zheng", cursive`（中文书法感）

### 空间系统
- 基础单位：8px
- 大量留白，模拟古籍版式
- 卡片间距：24px
- 内边距：32px（桌面）/ 16px（移动）

### 动效哲学
- 墨韵渐入：内容区块淡入 + 轻微上移（opacity 0→1, translateY 12px→0, 500ms ease-out）
- 笔触勾勒：按钮 hover 时边框如毛笔书写般延伸
- 翻页暗示：Tab 切换时内容区有轻微横向位移

### 视觉资产
- 图标：Lucide Icons（线条风格，stroke-width: 1.5）
- 装饰：CSS 绘制的抽象水墨晕染效果、印章风格标签
- 无需图片占位 —— 古籍图片由用户提供或内置素材

├─────────────────────────────────────────────────────┤

│  ┌────────┐ ┌────────┐ ┌────────┐                   │

### 响应式策略
- 桌面（>1024px）：2×3 网格卡片
- 平板（768-1024px）：3×2 网格
- 手机（<768px）：单列堆叠

## 4. Features & Interactions

### 核心功能

#### 4.1 内置古籍选择
- 下拉选择器展示内置古籍列表（名称 + 朝代 + 缩略图）
- 选择后自动填充原文文本和配图
- 内置古籍：诗经·关雎、论语·学而、庄子·逍遥游、史记·货殖列传（各提供原文 + 插图描述）

#### 4.2 手动输入模式
- 原文输入框（多行，支持粘贴）
- 配图描述输入框（文本描述插图内容，模拟上传）
- 标题输入框

#### 4.3 一键生成
- 点击"开始生成"按钮
- 显示墨韵加载动画（3个渐变圆点如水墨晕染）
- 依次生成6项内容（模拟 AI 逐项生成，每项300-800ms间隔）

#### 4.4 输出内容

| 卡片 | 内容说明 |
|------|----------|
| 白话解读 | 将古籍原文转化为通俗易懂的现代白话文 |
| 传播要点 | 3-5个适合社交媒体传播的核心亮点 |
| 二创边界 | 明确指出哪些改编是允许的，哪些是禁忌（如需保留原意的部分） |
| 短视频分镜 | 60秒内短视频的6-8个分镜描述（含画面、旁白、时长） |
| 视频提示词 | 用于 AI 生成视频/配图的英文提示词 |
| 分享卡 | 可保存为图片的社交媒体分享文案（含标题 + 要点 + 话题标签） |

#### 4.5 交互细节
- 每张卡片右上角有"复制"按钮，hover 时显示复制图标
- 分享卡支持"保存图片"功能（使用 html2canvas 将卡片导出为 PNG）
- Tab 切换"古籍选择"和"手动输入"时有墨韵过渡动画
- 输入区 empty state：显示淡灰色占位提示文字

### 边界情况
- 未选择/输入素材时点击生成：输入框抖动 + 红色边框闪烁
- 生成中再次点击：忽略（按钮显示 loading 状态）
- 所有输出卡片可独立滚动，内容过长时内部滚动

## 5. Component Inventory

### 5.1 Header
- 左：印章风格 Logo（"古籍活化"四字）
- 中：主标题 + 副标题
- 右：大赛参赛作品标签（朱砂红底白字）
- 状态：固定顶部，滚动时添加底部投影

### 5.2 古籍选择器
- 默认：显示"选择古籍..."（青铜色）
- Hover：下拉箭头旋转
- Open：卡片式下拉列表，带阴影
- Selected：显示选中古籍名称 + 朝代标签

### 5.3 输入卡片
- 标题栏：左侧标签 + 右侧可选操作
- 输入区：淡灰边框，focus 时变为朱砂红边框
- Textarea 自动增高（最大 200px）

### 5.4 生成按钮
- 默认：朱砂红背景，宣纸白文字，圆角 8px
- Hover：背景加深 10%，轻微上浮（translateY -2px）
- Loading：文字替换为三个墨点动画
- Disabled：灰色背景，不可点击

### 5.5 输出卡片
- 卡片标题：书法风格标签 + 图标
- 内容区：正文排版，支持 markdown 渲染
- 复制按钮：右上角，icon-only，hover 时显示 tooltip
- 分享卡特殊样式：金色边框，仿纸质卡片效果，底部有保存按钮

### 5.6 Footer
- 大赛信息：主办方、赛事周期
- 版权声明
- 背景：淡墨色，与头部呼应

## 6. Technical Approach

### 技术栈
- 纯前端单页应用（HTML + CSS + JavaScript）
- 无框架依赖，便于直接打开运行
- 使用 CSS Grid + Flexbox 布局
- 使用 CSS 变量管理主题色
- 使用 html2canvas（CDN）实现分享卡导出

### 内容生成
- 使用预设模板 + 关键词提取模拟 AI 生成
- 内置生成逻辑：解析输入文本，根据模板生成对应内容
- 分享卡使用 Canvas 或 html2canvas 导出

### 兼容性
- 支持现代浏览器（Chrome、Firefox、Safari、Edge 最新版）
- 无需 IE 支持
