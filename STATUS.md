# 古籍活化工坊 · 项目状态

> 当前版本：**1.7.1** | 更新日期：2026-07-04

---

## 产品形态

单一"可信传播案卷"首页（水墨册页调性）：案卷顶栏（案卷名+4场景tab+设置齿轮） + 左栏卷宗（古籍原文+来源+版本+原文链接） + 右栏产物（口播稿/封面/分镜/发布标题/引用说明） + 案卷底栏来源声明 + 阅读器工作台 + GSAP 入场/切换动画

### 核心流程
首屏看单一案卷 → 顶栏切换场景 tab（短视频/教学/文博/文创）→ 右栏产物同步更新 → 主题输入刷新案例 → 下方"进入创作"选文生成

### 产物列表（案卷右栏）
**口播稿**（考据）· **金句封面/海报**（演绎）· **60秒分镜**（考据）· **发布标题/标签**· **引用说明**

### 可信标识（仅 2 类）
**考据**（绿，基于原文）· **演绎**（朱砂，创作发挥）

## API 预留

- `POST /api/case/generate` — 生成全场景案例
- `POST /api/image/generate` — 生成单张图片
- `POST /api/video/generate` — 生成视频
- `GET /api/jobs/:id` — 查询任务状态
- 四态：idle → generating → completed / failed
- 无后端时自动降级 demo 数据，页面不坏
