> 当前版本：**1.5.0** | 更新日期：2026-07-04

## 产品形态

场景化产品叙事首页：首屏场景叙事（产品主张+用户任务+可信来源+生成预览+CTA） + 四类用户场景入口 + 产品指标 + 传播包（文字/图片/视频/发布/引用） + 阅读器工作台

### 核心流程
首屏看场景叙事 → 选择用户类型（短视频/教师/文博/文创） → 查看传播包 → 主题输入刷新 → 下方"进入创作"选文生成

### 产物列表（传播包结构）
**文字包**：白话解读（口播稿/课堂讲解/导览文案/商品文案）、传播要点、小红书文案
**图片包**：金句卡（CSS占位）、竖版海报（CSS占位）、图片 Prompt
**视频包**：60秒分镜时间轴、旁白稿、视频 Prompt
**发布包**：发布标题、标签
**引用说明**：标准引用格式、出处标注规则

## API 预留

- `POST /api/case/generate` — 生成全场景案例
- `POST /api/image/generate` — 生成单张图片
- `POST /api/video/generate` — 生成视频
- `GET /api/jobs/:id` — 查询任务状态
- 四态：idle → generating → completed / failed
- 无后端时自动降级 demo 数据，页面不坏
