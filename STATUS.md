## 产品形态

"摊开册页案台"可信传播案卷首页（古籍案台调性）：案卷顶栏（案卷名+4场景tab+设置齿轮） + 左栏原文摘录（题签式原文+来源+版本+原文链接） + 右栏主案例预览卡（传播成品感：封面+口播稿摘要+分镜缩略+朱砂印章CTA） + 5类签条式折叠传播包（文字/图片/视频/发布/引用） + 案卷底栏来源声明 + 阅读器工作台 + GSAP 入场/切换动画

### 视觉特征
- 去硬边框：全部 border 替换为淡墨渐变分隔线（linear-gradient 两端透明）
- 纸张层次：三层 box-shadow 渐进模拟纸张堆叠，米色径向渐变模拟纸纹
- 折角与签条：主案例卡右上角三角折角，左侧朱砂渐变条作题签
- 签条式折叠包：左朱砂竖条 + 悬停加浓，从"后台列表"变"册页签条"
- 印章式 CTA：ZCOOL XiaoWei 字体 + letter-spacing + 内描边 + 朱砂实色

### 核心流程
首屏看主案例预览卡 → 点击 CTA 展开完整传播包 → 顶栏切换场景 tab（短视频/教学/文博/文创）→ 主案例 + 产物同步更新 → 主题输入刷新案例 → 下方"进入创作"选文生成

### 主案例预览卡（右栏首屏）
**封面图**（朝代+书名+印章）· **口播稿标题+摘要**（80字）· **分镜缩略**（前3个）· **CTA 按钮**（生成完整传播包，一键展开5类）

### 折叠传播包（5类，默认收起）
**文字包**（白话解读/传播要点）· **图片包**（金句卡/海报/prompt）· **视频包**（60秒分镜）· **发布包**（标题/标签）· **引用说明**

### 可信标识（仅 2 类）
**考据**（绿，基于原文）· **演绎**（朱砂，创作发挥）

## API 预留

- `POST /api/case/generate` — 生成全场景案例（后端服务模式）
- OpenAI 兼容 `/v1/chat/completions` — 文字 LLM 直连模式（方案A）
- `POST /api/image/generate` — 生成单张图片
- `POST /api/video/generate` — 生成视频
- `GET /api/jobs/:id` — 查询任务状态
- 四态：idle → generating → completed / failed
- 无后端时自动降级 demo 数据，页面不坏

### 方案 A 安全边界
- API Key 仅存 localStorage，不上传服务器
- 适合 demo / 参赛验证场景
- 不作为生产级密钥托管方案
- 生产环境应改为服务端代理 + 密钥管理服务
