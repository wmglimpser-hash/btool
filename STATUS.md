> 当前版本：**1.8.0** | 更新日期：2026-07-05

## 产品形态

"主案例优先"可信传播案卷首页（水墨册页调性）：案卷顶栏（案卷名+4场景tab+设置齿轮） + 左栏卷宗（古籍原文+来源+版本+原文链接） + 右栏主案例预览卡（封面+口播稿摘要+分镜缩略+CTA） + 5类折叠传播包（文字/图片/视频/发布/引用） + 案卷底栏来源声明 + 阅读器工作台 + GSAP 入场/切换动画

### 核心流程
首屏看主案例预览卡 → 点击 CTA 展开完整传播包 → 顶栏切换场景 tab（短视频/教学/文博/文创）→ 主案例 + 产物同步更新 → 主题输入刷新案例 → 下方"进入创作"选文生成

### 主案例预览卡（右栏首屏）
**封面图**（朝代+书名+印章）· **口播稿标题+摘要**（80字）· **分镜缩略**（前3个）· **CTA 按钮**（生成完整传播包，一键展开5类）

### 折叠传播包（5类，默认收起）
**文字包**（白话解读/传播要点）· **图片包**（金句卡/海报/prompt）· **视频包**（60秒分镜）· **发布包**（标题/标签）· **引用说明**

### 可信标识（仅 2 类）
**考据**（绿，基于原文）· **演绎**（朱砂，创作发挥）

## API 预留

- `POST /api/case/generate` — 生成全场景案例
- `POST /api/image/generate` — 生成单张图片
- `POST /api/video/generate` — 生成视频
- `GET /api/jobs/:id` — 查询任务状态
- 四态：idle → generating → completed / failed
- 无后端时自动降级 demo 数据，页面不坏
