# 古籍活化工坊 · 项目状态

> 当前版本：**1.4.2** | 更新日期：2026-07-04

---

## 产品形态

案例驱动首页：首屏案例摘要（文字+图片+视频三类） + 主题输入 + 展开详情 + 阅读器工作台（"进入创作"）

### 核心流程
首屏看案例摘要 → 输入主题刷新案例 → 展开查看详情 → 下方"进入创作"选文生成

### 产物列表（按三类组织）
**文字类（P0·前端壳+API预留）**：白话解读、传播要点、小红书文案
**图片类（P0·前端壳+API预留）**：金句卡（CSS占位）、竖版海报（CSS占位）、图片 Prompt
**视频类（P0·前端壳+API预留）**：60秒分镜时间轴、旁白稿、视频 Prompt、发布标题/标签

## API 预留

- `POST /api/case/generate` — 生成全场景案例
- `POST /api/image/generate` — 生成单张图片
- `POST /api/video/generate` — 生成视频
- `GET /api/jobs/:id` — 查询任务状态
- 四态：idle → generating → completed / failed
- 无后端时自动降级 demo 数据，页面不坏
