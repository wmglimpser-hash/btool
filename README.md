# 古籍活化工坊

> 让古籍内容更容易被准确传播，而不是被随意曲解

一个面向文博、教育、科普和短视频创作者的古籍活化工具。我们坚持：**活化不能脱离原文，传播必须遵循史实**。

## 项目简介

选择一页古籍，系统基于原文生成可传播内容：

- **活化溯源**：明确典籍版本、篇目、参考注本与时代背景
- **白话解读**：将古籍原文转化为通俗易懂的现代白话文，逐句标注原文锚点
- **传播要点**：提炼适合社交媒体传播的核心亮点
- **二创边界**：用绿区/红线可视化改编边界，守护文化本真
- **短视频分镜**：生成 60 秒短视频的横向时间轴分镜脚本，每镜标注原文依据
- **视频提示词**：提供可直接用于 AI 视频生成的英文提示词
- **文创分享卡**：生成带出处水印的竖排社交媒体分享卡片

## 反伪科普设计

本项目内置三层约束机制，防止古籍活化沦为伪科普：

1. **原文锚定**：白话解读与分镜脚本均标注对应原文
2. **出处溯源**：每部内置古籍明确版本、参考注本与时代背景
3. **置信度评分**：高/中/低置信内容分区提示，低置信部分强制改编声明

## 使用方式

1. 在「内置古籍」中选择一本典籍，或切换到「手动输入」粘贴自己的古籍内容
2. 点击「开始活化」按钮
3. 查看生成结果，重点关注顶部的「活化溯源」卡片
4. 展开「原文锚点」核对白话解读与原文的对应关系
5. 下载分享卡或复制引用格式进行传播

## 技术栈

- 纯前端单页应用（HTML + CSS + JavaScript）
- 无需后端，可直接在浏览器中运行
- 使用 html2canvas 实现分享卡片导出
